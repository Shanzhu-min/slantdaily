# 题库设计目的
- 在页面加载时，通过seed直接调取题目，降低页面加载时的损耗。
- 从题库加载的题目，能够保证有解。
- 从题库加载题目，能够记录每一道题的加载次数、解题成功次数。为后续的题库分级提供数据基础。

# 题库难度分级
- 题库中难度分为easy/medium/hard三个级别。
- 题目的评分标准是：difficulty_score =  0.25 * (1 - forced_ratio)+ 0.20 * branching_factor+ 0.20 * inference_depth+ 0.15 * (1 - clue_density)+ 0.15 * structure_score+ 0.05 * guess_required
- easy 题目的标准是：forced_ratio > 0.65、branching < 1.3、depth ≤ 3、density > 0.55、structure_score < 0.3
- medium题库的标准是：orced_ratio 0.45–0.65、branching 1.3–2.0、depth 4–7、density 0.40–0.55、structure_score 0.3–0.6
- Hard题库的标准是：forced_ratio < 0.45、branching > 2.0、depth ≥ 8、density < 0.40、structure_score > 0.6
- 各难度的格子数分别是：easy的格子数是6*6、medium的格子数是8*8、hard的格子数是10*10；
