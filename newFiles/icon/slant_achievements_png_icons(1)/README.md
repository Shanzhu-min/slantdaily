# Slant Achievements PNG Icons

This package contains production-ready PNG achievement icons for Slant Game Online.

## Contents
- achievements.json
- icons/png/locked/*.png
- icons/png/unlocked/*.png

## Icon Specs
- 15 achievements
- 2 states per achievement
- 30 PNG files total
- 512 x 512 px
- Locked icons: dark gray theme
- Unlocked icons: colorful theme

## Suggested Usage
Place the `icons` folder and `achievements.json` under your Next.js `public` folder.
